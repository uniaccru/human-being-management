# HumanBeing Management System

Веб-информационная система для управления объектами HumanBeing с поддержкой CRUD-операций, массового импорта данных и работы в многопользовательской среде.

---

## Cтек

- Jakarta EE, CDI
- JPA (EclipseLink)
- PostgreSQL
- Druid Connection Pool
- Ehcache (L2 JPA Cache) + CDI Interceptors
- MinIO
- Apache JMeter (нагрузочное и конкурентное тестирование)

---

## Функциональность

- CRUD для HumanBeing и связанных сущностей
- Табличный интерфейс с пагинацией, сортировкой и фильтрацией
- Реал-тайм синхронизация изменений между пользователями
- Валидация ограничений на уровне ORM, БД и бизнес-логики
- Специальные бизнес-операции (группировка, массовое удаление/обновление, поиск по подстроке)
- Массовый импорт объектов из файла с полной транзакционностью
- Лог истории импорта с разграничением прав доступа
- Хранение импортируемых файлов в MinIO с возможностью скачивания

---

## Транзакции 

- Транзакционный импорт с откатом при любой ошибке
- Распределённая транзакция БД + файловое хранилище  
  (реализация 2PC на уровне бизнес-логики)
- Поддержка конкурентных операций (создание, обновление, удаление, импорт)

---

## Производительность

- Connection Pool (Druid)
- L2 Cache (Ehcache) с возможностью логирования cache hits/misses
- Нагрузочное тестирование и сценарии конкурентного доступа в Apache JMeter
